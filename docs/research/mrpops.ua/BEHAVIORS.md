# Mr. Pops Behaviors

- Header is fixed at the top and overlays the hero.
- The page sets `use-smooth-scroll` on the root HTML class after load.
- Hero uses an animated preloader before revealing the page.
- Primary CTAs and product cards use hover transitions and button state styling from the original site bundles.
- Product cards expose add-to-cart actions inside the card layout.
- Gift box section includes embedded video media.
- Footer contains a topic dropdown plus form inputs and submit CTA.
- The runtime is an older Next.js pages-router app hydrated from bundled client scripts and `__NEXT_DATA__`.
